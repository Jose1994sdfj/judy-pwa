¿Cuántos azulejos necesito para un baño de X m²?

Fórmula exacta, tabla de referencia y calculadora interactiva para que no te sobren ni te falten piezas. Actualizado 2026.

Respuesta rápida

Para calcular azulejos: divide el área a revestir en m² entre el área de cada pieza en m², suma 10% de desperdicio. Un baño de 4 m² de pared requiere aproximadamente 20–24 m² de revestimiento (incluyendo paredes y piso). Con piezas de 30×30 cm (0.09 m²/pieza) necesitas ~265 piezas.

Fórmula paso a paso

1. Calcula el área de muros: Perímetro × altura de muros. Resta huecos de puerta y ventana (aprox. 1.8 m² por puerta estándar).

2. Suma el área del piso si también lo vas a revestir: largo × ancho.

3. Divide entre el área de la pieza (ej. una pieza de 30×30 = 0.09 m²).

4. Suma el porcentaje de desperdicio: 10% para cortes rectos, 15% para diagonal.

5. Redondea siempre hacia arriba y compra de una sola partida (mismo lote) para evitar diferencias de tono.
